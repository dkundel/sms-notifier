# sms-notifier
An SMS notification system using Twilio for Berlin Tech Open Air

1. Clone the repo:
```sh
git clone git@github.com/dkundel/sms-notifier.git
cd sms-notifier
```

2. Copy Config
```sh
copy _config.js config.js
```

3. Fill out fields in config.js
4. Install dependencies
```sh
npm install
```
5. Start server:
```sh
npm start
```
